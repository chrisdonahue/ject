JECT

JECT (JUCE Extended Convolution Techniques) is a simple, cross-platform demo application for experimenting with extended convolution techniques as described in the Extended Convolution Techniques for Cross-Synthesis ICMC 2016 paper. It supports WAV, AIFF and OGG files of arbitrary length.

To use JECT, drag two files (either separate or simultaneously) onto the window. Press Convolve to perform convolution on the two sounds with the current parameter configuration. Press Play or Loop to hear the results. Adjust the Gain or NFFT (output length) with the top sliders. P/R Behavior determines the strategy for coupling the P and R parameters in the UI. The P, R, Q, S sliders control algorithm behavior as defined in the paper. The algorithm is not real-time so you will need to press Convolve to hear the changes.

Building

We are currently not distributing binaries for this software, though this may change in the future.

  1. Download JUCE 4.2.1
  2. Compile the Projucer under extras/Projucer
  3. Download JECT
  4. Open ject.jucer in Projucer
  5. Under Config>JECT>Modules, change the module paths to point to the modules folder inside JUCE.
  6. In Projucer, File>Save Project and Open in IDE
  7. Compile using your IDE or make if on Linux
